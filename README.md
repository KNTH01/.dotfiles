# dotfiles
My dotfiles

Add `~/.gitconfig.user` with you Git credentials

```plaintext
// ~/.gitconfig.user
[user]
  name = My name
  email = my-email@gmail.com
```

```bash
ln -s ~/dotfiles/.gitconfig ~/.gitconfig
```
